Technical Specifications

  • Inductance Value: Measured in Henrys (H), it indicates the inductor's ability to store energy. Common values include 10µH, 100µH, 1mH, etc.
  • Core Material: The type of core (Air-core, Iron-core, Ferrite-core) affects the inductor's performance and applications.
  • Current Rating: Indicates the maximum current the inductor can handle without saturation or damage.
